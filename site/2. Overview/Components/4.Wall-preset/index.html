    Wall Preset¶

    1. What is a Wall Preset?¶

    A Wall Preset is a Struct that defines how a generated SplineArchitectWall actor will look. It's located in the Preset category of a SplineArchitectWall actor (in the Details panel).

    2. What do the Wall Preset parameters do?¶

    FloorSurfaces parameters (FWallPreset):¶

    Parameter Example
    WallHeight
    ‚Äì Sets the height of the wall; this value determines the scale for all meshes.
    HeightOverride
    ‚Äì If true, allows manual override of wall height; otherwise, WallHeight is taken from the first WallMeshes array element.
    WallLength
    ‚Äì Sets the length of the wall; spline segments will be divided according to this value.
    LengthOverride
    ‚Äì If true, allows manual override of wall length; otherwise, WallLength is taken from the first WallMeshes array element.
    CornerLength
    ‚Äì Sets the corner length; this value is subtracted from segments to form a corner.
    CornerLengthOverride
    ‚Äì If true, allows manual override of corner length; otherwise, it's taken from CornerMesh.
    RandomizeSegmentsBy
    ‚Äì Randomly alters segment lengths up to the given threshold.
    WallZHandling
    ‚Äì Determines how to handle Z-axis height differences between segments (KeepZ, FollowSpline, Skew).
    SkipPieces
    ‚Äì Allows skipping specific segments (straight or angled).
    Inset
    ‚Äì Offsets meshes inward or outward from the spline by a given distance.
    OverlapPiecesBy
    ‚Äì Overlap size to prevent gaps caused by imperfect mesh edges.
    NumFloors
    ‚Äì Number of times to duplicate the segments vertically with offset equal to WallHeight (number of floors).
    WallMeshes
    ‚Äì Main meshes for wall segments; the best fit is chosen based on segment length.
    FillerWallMeshes
    ‚Äì Additional meshes used to fill gaps so that the main WallMeshes don't stretch or shrink.
    MinFillerWallLength
    ‚Äì Minimum length for filler meshes; shorter meshes won‚Äôt be used.
    KeepWorldScale
    ‚Äì If true, meshes are not scaled to fit the segment, and retain their original scale.
    WorldScale
    ‚Äì Original scale applied to meshes when KeepWorldScale is true.
    CornerMesh
    ‚Äì Mesh used to form corners.
    CornerType
    ‚Äì Type of corner: Straight, Bend, Chamfer, or Pipe.
    OffsetBy
    ‚Äì Offsets all meshes by a given vector.
    RandomizeLocationBy
    ‚Äì Randomizes mesh positions within the given vector (recommended small values to avoid Z-fighting).
    FloorSurfaces
    ‚Äì Array of floor surface configurations; each element defines the parameters of a floor surface.
    PostConfigurations
    ‚Äì Array of post configurations determining how and where posts will be generated.
    RandomisationSettings
    ‚Äì Randomization settings for mesh distribution and transforms.
    MaterialParameters
    ‚Äì Array of material parameters allowing customization of specific material properties on certain meshes.

    Parameter Example
    Material
    ‚Äì Material applied to the floor surface.
    AlignTo
    ‚Äì Floor alignment relative to wall: Top, Center, or Bottom.
    ZOffset
    ‚Äì Vertical offset for floor placement.
    Inset
    ‚Äì Inward or outward offset of floor from the wall contour.
    Thickness

    ‚Äì Controls polygon count of the floor.

    ColorBlurIterations
    ‚Äì More iterations = smoother vertex color blending.
    UseDelaunayTriangulation
    ‚Äì Enables Delaunay triangulation for better mesh quality. If the floor mesh doesn't work, try disabling this.

    PostConfigurations parameters (FPostData):¶

    Parameter Example
    PostMesh
    ‚Äì Mesh used for generating posts.
    GenerationMethod
    ‚Äì Method for placing posts: OnCorners, OnSplinePoints, BetweenSegments, OnCustomPieces, Periodically, StartAndEnd.
    AverageDirections
    ‚Äì Used only when GenerationMethod is OnSplinePoints. Averages directions.
    PeriodDistance
    ‚Äì Used when GenerationMethod is Periodically. Defines distance between posts.
    PostOffset
    ‚Äì Offsets posts by a vector (X = along spline, Y = inward/outward, Z = vertical).
    AddRotation
    ‚Äì Adds rotation to all posts.
    bScaleToWallHeight
    ‚Äì Automatically scales post to match WallHeight.
    PostScale
    ‚Äì Adjusts the scale of the post.
    SkipRule
    ‚Äì Can skip posts if the segment is flat or angled.

    RandomisationSettings parameters (FRandomSettings):¶

    Parameter Example
    RandomMeshes
    ‚Äì Array of FRandomMeshes structs defining random replacement rules for meshes.

    FRandomMeshes parameters:¶

    Parameter Example
    KeyMesh
    ‚Äì Target mesh to be randomly replaced by one from the RandomMeshes array (including itself).
    RandomMeshes
    ‚Äì Array of meshes with weights (FRandomMeshWithWeight), from which a random one is selected. Higher weight = higher chance.
    FitRandomType
    ‚Äì MatchScale or KeepScale ‚Äì whether to scale the mesh to fit or keep its original size.
    Knockout
    ‚Äì Controls how many meshes to randomly remove (0.0f = none, 1.0f = all).
    MirrorX/Y/Z
    ‚Äì Allows scaling by -1 on specific axes. Only works when converting to StaticMeshComponents.
    Rot90
    ‚Äì Randomly rotates meshes by 90, 180, or 270 degrees.
    Rot180
    ‚Äì Randomly rotates meshes by 180 degrees only.

    MaterialParameters parameters (FMatParams):¶

    Parameter Example
    MaterialApplicationMode
    ‚Äì Application mode: Include (only listed), Exclude (all except listed), or All.
    MaterialReferences
    ‚Äì List of materials to apply parameters to (when mode is not All).
    MeshApplicationMode
    ‚Äì Mesh selection mode: Include, Exclude, or All. Lets you change material parameters on specific meshes.
    StaticMeshReferences
    ‚Äì List of meshes to apply parameters to (when mode is not All).
    Parameters
    ‚Äì Array of material parameters (FMatParam).

    Parameter Example
    ParameterName
    ‚Äì Name of the material parameter to modify.
    ParameterType
    ‚Äì Type of parameter: Float or Vector.
    FloatValue
    ‚Äì Value when type is Float.
    ColorValue
    ‚Äì Value when type is Vector (color).
